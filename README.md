# dotfiles

Sane dotfiles for an Arch user, providing configuration for the following programs:

- Neovim
- AwesomeWM
- WezTerm
- Lush
- Picom
- Joshuto
- OneDrive
- Neofetch
- Stylua

## Installation

If for some reason you're crazy enough to want to install all of my dotfiles, you can run the install script. This is mostly intended for my own use on freshly installed systems. **Note that this will only work properly on Arch linux (and possibly other Arch-based distros, but untested).**
